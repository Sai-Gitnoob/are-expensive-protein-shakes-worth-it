# 🥤 Protein Drink Market Analysis.

A **data science project analyzing popular protein drinks in the consumer market** using data analytics and visualization techniques.

The goal of this project is to build a **structured dataset of well-known protein beverage brands** and explore insights such as:

- Protein content comparison  
- Price vs protein value  
- Nutritional composition  
- Brand distribution in the market  

This project demonstrates **data collection, cleaning, analysis, and visualization**, making it a practical example of **real-world consumer data science**.

---

# 📌 Project Objective

The objective of this project is to:

1. Identify **popular protein drink brands** available in the consumer market.
2. Build a **clean structured dataset** containing important attributes.
3. Perform **exploratory data analysis (EDA)**.
4. Generate **visual insights about the protein drink market**.
5. Create **informative visualizations** to compare nutritional and economic value.

---

# 📊 Dataset Features

The dataset includes several important attributes for each protein drink.

| Feature | Description |
|------|------|
| Brand Name | Company or brand producing the drink |
| Product Name | Specific product variant |
| Protein (g) | Protein content per serving |
| Calories | Energy per serving |
| Sugar (g) | Sugar content |
| Fat (g) | Total fat |
| Serving Size | Quantity per serving |
| Price | Market price |
| Price Unit | Price for the specific quantity |
| Protein per Rupee | Value metric |
| Country of Origin | Manufacturing country |

# 🔍 Key Questions Explored

This project aims to answer several **market and nutrition-related questions**:

- Which protein drinks provide **the most protein per serving**?
- Which brands offer the **best protein value for money**?
- What is the **relationship between protein content and price**?
- How do brands differ in **sugar and calorie content**?
- Which brands dominate the **consumer protein drink market**?
